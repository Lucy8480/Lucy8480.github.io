2. 결과 도출

[##Image|kage@btqSCG/btruqEnMze0/AAAAAAAAAAAAAAAAAAAAADpv_5NUbUBLc1Grsp-V_ywQaxYtHYRjig2rHsoPvX85/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1756652399&allow_ip=&allow_referer=&signature=eChXoKAJOZlMTNgT5sZd7ar7LL8%3D|CDM|1.3|{“originWidth”:550,”originHeight”:170,”style”:”alignCenter”,”width”:586,”height”:181}##]

<그림.1> int형 4바이트씩 끊어서 표현한 for 과 res 데이터

res 값은 for문의 값을 5번 누적한 결과.

res 값을 5로 나누어봤을때 for에서 쓰인

데이터는 한번당 0x6C5CEC8입니다.

이것을 다시 5로 곱해지면 0x4가 부족.

데이터는 0x6C5CEC8 4개와

0x6C5CECC의 데이터를

더하는 것으로 가정을 하고

프로그램에 입력값으로 넣어주면 됩니다.

[##Image|kage@cISDmB/btrunO5G7OP/AAAAAAAAAAAAAAAAAAAAAG9RFFP6I__ubjI0x0QEEOAqlpxFEB9Dbw06XVOG3DT4/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1756652399&allow_ip=&allow_referer=&signature=DPpWC9QIiXMqJyXlgg4o0sLi330%3D|CDM|1.3|{“originWidth”:550,”originHeight”:28,”style”:”alignCenter”,”width”:838,”height”:42}##]

16진수로 데이터를 전달해주면 됩니다.
